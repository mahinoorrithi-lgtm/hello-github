# File Upload

## Payload Used

```php
shell.php
```

## Description

File Upload vulnerability allows attackers to upload malicious files to the server.

## Steps

1. Create a file named:
   ```php
   shell.php
   ```
2. Save file as:
   ```text
   All Files (*.*)
   ```
3. Open DVWA
4. Navigate to File Upload
5. Upload the PHP shell file successfully

## Result

Successfully uploaded malicious PHP file to the server.
